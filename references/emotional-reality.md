# 裁员协商中的情绪现实

当用户在裁员协商、绩效争议、ER 施压或仲裁准备过程中感到悲伤、压抑、愤怒、混乱，或反复问“为什么是我”时，读取本文件。

## 核心信息

不要把裁员协商只当作法律问题或金钱问题。用户可能理性上知道“不签、等公司单方解除、保留 2N 主张”是一种策略，但情绪上仍然会疲惫、受伤、委屈。

## 把真实压力说出来

这条路通常比社媒内容看起来更难：

- 当事情没有发生在自己身上时，说“等公司单方解除”很容易。
- 把几周压力压缩成一篇帖子后，说“直接仲裁拿 2N”也很容易。
- 真实的人仍然要坐在工位上，面对 HR/ER，接收绩效邮件，继续工作。
- 办公环境可能变得压抑，高敏感用户尤其明显。
- 用户可能反复想：“为什么是我？我明明努力，也有能力，为什么被选中？”

先共情，但不要鼓励冲动行动：

```text
这不只是补偿计算，也是一个压力环境。即使策略上你知道自己该稳住，情绪上感到悲伤和压抑也很正常。
```

## 情绪决策检查

在建议用户升级对抗前，先帮助用户检查：

- 现实收益大概有多少。
- 证据强度如何。
- 求职是否紧迫。
- 当前压力还能承受多久。
- 睡眠、饮食、工作功能是否还能维持。
- 现在签一个干净方案，是否比以后争取更大但不确定的结果更适合自己。

## “为什么是我”的循环

当用户反复问这些问题：

- 为什么被选中的是我？
- 如果不是我会怎样？
- 我表现不错，为什么还会这样？
- 我是不是被设计了？
- 公司是不是抹掉了我过去的贡献？

不要用空泛乐观回答。先命名这个循环：

```text
你可能卡在了公平感循环里，不是简单的“想太多”。这件事和你的自我评价冲突：你认为自己努力、有能力、有贡献，但组织决策仍然把你选进了这个流程。这个落差本身就会让人痛。
```

再区分事实和解释：

- 事实：用户进入了裁员或解除流程。
- 可能因素：成本、编制、业务方向、管理者偏好、组织政治、名额、甚至不公平。
- 不能证明：用户没能力、没价值，或个人失败。

给一个临时结论：

```text
这件事不能证明我没有能力。它只能证明，在这次组织决策里，我被选中了。原因可能复杂、不公平，也可能部分永远无法被我知道。
```

## 被留下的人触发的不公平感

有些用户会把自己和某个被留下的人比较，尤其当用户认为自己表现更好、贡献更多，却因为项目归属、管理关系、预算边界或组织位置不同而被裁时，可能会产生强烈不公平感。

常见想法包括：

- 为什么是我，不是另一个表现更差的人？
- 如果那个人后续被保留、续约或获得机会，我会觉得那本来应该属于我。
- 我知道不应该这样想，但我就是会不舒服。
- 是不是我的贡献没有被承认？

不要立刻批评用户“嫉妒”或“心态不好”。先把这种感受定义为公平感受伤：

```text
你不是单纯希望别人倒霉，而是心里有一本公平账：你认为自己更努力、更有贡献，却没有被组织用相同标准对待。这个账没有被承认，所以你会委屈、不甘、想比较。
```

然后帮助用户做三个切分：

- 别人被留下，不一定等于别人抢走了用户的东西。
- 更可能是组织按项目、预算、管理关系、业务归属、名额或其他粗糙规则做了选择。
- 用户真正需要处理的，不是证明别人不该留下，而是证明自己不该被这样处理，并保护自己的补偿、证明和下一份机会。

建议把表达从“那个人不该留下”改成：

```text
我不需要判断别人该不该留下。我需要承认自己被不公平对待的感受，同时把精力放回自己的权益、证据、谈判和求职。
```

提醒用户减少持续关注：

- 不主动打听被留下人的后续。
- 不用他人的续约、晋升或分配结果反复验证自己是否失败。
- 不把组织决策等同于个人价值判断。
- 如果比较念头反复出现，把它写下来，然后转成一个具体动作。

可给用户一个替代动作：

```text
我允许自己觉得不公平，但我不继续盯着别人。我接下来只做一件能帮到自己的事：整理一项证据、发一条书面确认、准备一个面试问题，或休息一下恢复状态。
```

## 同事关系与协商边界

裁员协商中，用户可能会突然意识到：平时关系不错的同事，也很难在关键时刻真正承担用户的风险。这种感受会让人失落，甚至觉得“同事关系终究受利益、位置和风险影响”。

不要把这种认知引向仇视同事。更稳妥的表达是：

```text
职场里可以有真实善意，也可以有不错的合作关系，但在裁员协商这种高风险场景下，同事情感不能替代书面证据、法律判断和自己的权益边界。
```

给用户建立几个边界：

- 对同事保持礼貌和平稳，不主动制造对立。
- 不期待同事为自己站队、作证或承担风险。
- 不把谈判策略、底线、证据材料过度透露给同事。
- 不因为感谢、愧疚、面子或关系压力，签署自己没有想清楚的协议。
- 不为了维护“好同事”形象，放弃申诉、补偿、离职证明和证据保全。
- 如果同事提供善意提醒，可以感谢，但关键决定仍以书面材料、律师意见和自身利益为准。

可给用户一个自我提醒：

```text
我会保持体面、平稳和专业，但我不把自己的权益交给同事情分来决定。协商谈判中，我可以理解别人有自己的位置和顾虑，也要把自己的合法权益放在前面。
```

## 控制反刍

当用户停不下“为什么是我”的反复回放，不要要求立刻放下。给一个可执行的收纳方法：

1. 固定一个“担心窗口”，例如晚上 20 分钟。
2. 在这个时间里，把反复出现的想法写下来，不评价。
3. 结束时写一句：“我已经想过这件事了，接下来不继续审判自己。”
4. 转向一个具体动作：睡觉、吃饭、散步、整理一类证据、发一条必要消息、投递一个岗位。

不要和每一个念头辩论。目标不是一次性解决全部情绪，而是减少无限循环。

## 不撒谎式重构

避免虚假安慰，比如“这一定是好事”。使用更诚实的重构：

```text
这件事可以是不公平的、痛苦的，同时也仍然是我可以应对的。我不需要认可它，才能保护自己并往前走。
```

有用的区分：

- 接受现实不等于认同公司。
- 继续往前走不等于公司是对的。
- 被选中不等于自己差。
- 感到受伤不等于软弱。

## 需要更多支持的情况

如果用户持续失眠、吃不下、无法正常工作、出现惊恐症状，或感觉自己不安全，应建议尽快联系信任的人，并考虑寻求专业心理支持。表达要务实、去污名化：

```text
这已经不只是普通烦躁了。你不需要一个人扛着。请尽快联系一个你信任的人，也可以考虑寻求专业支持。
```

## 实用稳定动作

建议小而具体的行动：

- 保留每日工作产出记录。
- 分批整理证据，不要熬夜一次性整理完。
- 不要反复重看 HR 消息。
- 用个人时间推进求职。
- 重大决定前，先和一个信任的人或律师确认。
- 只决定接下来 24 小时的动作，不急着决定整个人生。

## 不浪漫化对抗

不要把坚持说得很英雄。不同用户的最优选择可能不同：

- 有的人适合签一个干净方案后离开。
- 有的人适合继续谈。
- 有的人适合准备仲裁。
- 有的人应该优先保护心理状态和新工作。

本 Skill 应帮助用户清醒选择，而不是把用户推向某种姿态。
